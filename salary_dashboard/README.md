# Payroll Salary Dashboard
An interactive payroll analytics dashboard built using Python and Streamlit.

# Features
- Employee-wise salary breakdown
- Statutory, voluntary & recovery analysis
- Company-wide salary distribution
- Benefit utilization & cost analysis
- Month-wise filtering

# Tech Stack
- Python
- Pandas
- Streamlit
- Matplotlib / Streamlit Charts

# To run code 
python -m streamlit run trial.py
